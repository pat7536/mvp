Add these two files. 2. In **Project Settings → Environment Variables**, add: * `OPENAI_API_KEY` = *your actual key* (rotate the one you posted earlier; don’t expose it in the client!). 3. **Deploy**. Open the site → `index.html` loads as the app. The **Draft with ChatGPT** button now calls your `/api/chat` and fills the lesson. > Optional: If you deploy elsewhere (Netlify/Cloudflare), rename the function as needed (Netlify `netlify/functions/chat.js`, Cloudflare Workers with `fetch` handler). The browser code stays the same—still calls `/api/chat`.
